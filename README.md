# 資格免許管理

**あなたの資格・免許を一元管理**

資格免許管理アプリは、取得した資格や免許証を簡単に記録・管理できるアプリです。有効期限の管理や更新時期の通知で、大切な資格の失効を防ぎます。

## 主な機能

### 📝 資格・免許の登録
取得した資格や免許証の情報を簡単に登録できます。資格名、取得日、有効期限などを記録して一元管理。

### 🔔 有効期限の通知
更新が必要な資格の有効期限が近づくと通知でお知らせ。更新手続きを忘れる心配がありません。

### 📊 一覧表示
登録した資格を見やすく一覧表示。カテゴリ別の整理や検索も可能です。

### 🔒 安全な管理
大切な資格情報をデバイス内で安全に管理。プライバシーを守りながら利用できます。

## こんな方におすすめ

- 複数の資格や免許を持っている方
- 資格の更新時期を忘れがちな方
- 就職・転職活動で資格情報をまとめたい方
- 業務上、資格管理が必要な方
